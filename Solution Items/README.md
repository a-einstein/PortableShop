## PortableShop

#### Purpose
* Testing and demoing a portable application based on the use of Xamarin.
* It should become an equivalent of the WpfShop.
* Follow the commits to see in what stage this is in.

#### Aspects
* Xamarin.
* MVVM.
* Client-server.
* WCF.
* Globalized resources.
* NuGet.
* ...
